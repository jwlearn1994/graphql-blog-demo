# GraphQL Blog Demo

this is a practice demo in using `apollo server` to create a fake graphql blog


## Install
```
$ npm install
```

create `.env` file with at least

```js
SALT_ROUNDS=2
SECRET='graphql_blog_secret'
```

## Run up
```
$ npm run dev
```
